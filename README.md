## About Me

I am Brian — a UC San Diego Mathematics–Computer Science graduate who builds practical, scalable software with a focus on mobile development, AI applications, and full-stack engineering.

My work includes:

- Mobile apps using React Native, Swift, and Android/Expo
- AI-driven tools such as resume optimization, ingredient recognition, and predictive modeling
- Machine learning and analytics projects (including NBA scoring models)
- Full-stack systems powered by FastAPI, PostgreSQL, Docker, and Firebase

I focus on:

- Designing systems that scale and evolve
- Delivering clean and thoughtful user experiences
- Iterating from MVP to production-ready solutions
- Applying AI where it meaningfully improves outcomes

## Featured Projects

**resuMAX — AI Resume Optimization Platform**  
A FastAPI-based system that analyzes resumes against job descriptions, generates improvement suggestions, and delivers tailored rewrite recommendations.

**Countdown Calendar (React Native + FastAPI)**  
A cross-platform mobile app for creating and tracking upcoming events, designed for future expansion into widgets and calendar integrations.

**Whats Cookin AI**  
A backend-driven tool that identifies ingredients and recommends recipes, combining structured data modeling with AI-supported logic.

**NBA Analytics and Prediction Model**  
An exploratory and predictive analytics project focused on player scoring outcomes, used for feature engineering and model experimentation.

**Habitizer — Mobile Habit Tracking App**  
An Android application built using Agile methodologies, featuring goal tracking, reminders, and user-centric interface design.
